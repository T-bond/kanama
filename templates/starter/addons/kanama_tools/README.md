# Kanama Tools

Kanama Tools is the optional editor plugin packaged with Kanama. It adds editor
commands for building Kotlin scripts, opening the configured Kotlin source
folder, and checking whether the desktop JVM can be found.

The Kanama runtime files live in `res://addons/kanama`. For setup instructions,
see `res://addons/kanama/README.md` in packaged addons or the Kanama
documentation.
